== Description ==

APARG Slider is image slider with descriptions for each image and smooth animations.
This plugin let users  create multiple sliders with descriptions for each slide.
You can create multiple sliders and embed slider in both template or post.
Each image can have multiple descriptions with custom colors.
There are a lot of settings that can customize  your slider.

For comments <a href='mailto:info@aparg.comm'>info@aparg.com</a>

Check demo <a href='http://www.aparg.com'>www.aparg.com</a>

== Installation ==

1. Upload `aparg-slider.zip` and unpack it to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add slider, afterwards add images with their descriptions
4. Place `<?php echo do_shortcode('[aparg_slider id=SLIDE_ID]'); ?>` in your templates or use [aparg_slider id=SLIDE_ID] for posts
